Overview

Analytics are managed by PLAN and can be accessed here. PLAN is installed on all the Minecraft servers. The web server is run internally from BungeeCord.

Encryption

PLAN requires encryption to be enabled for logins to work. There is no internal encryption (ie between the internal webserver and the tunnel), so PLAN's SSL setting is set to 'proxy', which trusts the proxy to encrypt all traffic (ie the tunnel).
