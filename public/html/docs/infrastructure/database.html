MariaDB

MariaDB will be phased out for game servers in favour of more reliable local storage. /etc/mysql sed -n '/^-- Current Database: `(database)`/,/^-- Current Database:/p' fulldump.sql > test.sql mysql -umc -p -o (databasename) < (filename.sql) mysql -umc -p -o

MariaDB is a fork of MySQL. We use it instead of MySQL because it provides some small additional functionality and is maintained as a FOSS fork in an effort to escape Oracle's control of MySQL. Currently, several plugins on Survival and Lobby use it. The test servers and Novacore do not use MariaDB. Note that both 'mysql' and 'mariadb' can be used interchangeably in commands.
