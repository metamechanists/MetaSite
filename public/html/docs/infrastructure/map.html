Overview

The web map is operated by the Squaremap plugin. This plugin manages an external folder, which is where the map data is stored. This folder contains a directory of the tiles, which is actively updated by Squaremap. The other folders concern the web frontend. Squaremap comes with its own default frontend, but we use LiveAtlas because it just looks much nicer. LiveAtlas is simply a set of html/css/js files that occupy the root folder, and provide an interface to the tiles. Some of the css for LiveAtlas has been modified to remove ugly highlighting.
