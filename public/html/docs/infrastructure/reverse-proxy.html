Overview

A reverse proxy takes incoming requests on different ports, and routes them to services internally. For example, if a request is made on port 8804, we could set up a reverse proxy to route it to an internal web server. The reverse proxy we use is nginx for its simplicity and speed. Nginx uses a set of configs, where each config file represents one service.
