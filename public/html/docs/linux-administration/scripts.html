Scripts

Local scripts

These scripts can only be run as regular shell scripts and cannot be called from any location as a command can.

/home/idra/MountNAS.sh /home/idra/start.sh

Aliased scripts

These scripts can be run from anywhere on the command line. This is done through aliasing in .bashrc.

startserver [server] show [server] createuser [user] renameuser [old-user] [new-user]
