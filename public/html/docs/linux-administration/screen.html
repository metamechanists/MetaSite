Screen

Screen allows for persistent terminal sessions to be started and navigated between. This is necessary for Minecraft servers, as we would ideally want to be able to access the console (which is much harder with systemctl etc).

  • Ctrl+a+d will detach from a session you're connected to
  • Ctrl+a+: will open the screen command line for the attached screen
  • Ctrl+a+esc will open copy mode (which allows you to scroll)

List your user's running sessions

screen -ls

Attach to an existing session

screen -r [name]

Create a new session

screen -S [name]

Kill an existing session

screen -r [name] -X quit

Enable multi-user

screen -S [name] -X multiuser

Add a user to a screen

screen -S [name] -X acladd [user]

Terminate ALL running screens

pkill screen
